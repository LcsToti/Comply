# Comply

![Logo](https://github.com/user-attachments/assets/f968c405-ceb5-4b39-90c6-2c7b420adfdd)



`Análise e Desenvolvimento de Sistemas`

`Desenvolvimento de uma Aplicação Distribuída`

`4º Semestre`

O projeto é um sistema de leilões online, onde usuários podem cadastrar produtos, dar lances ou comprar itens imediatamente. O sistema gerencia leilões, produtos, lances e transações, garantindo que usuários recebam notificações sobre as ações importantes e possam acompanhar o andamento dos leilões.

Ele também oferece recursos de administração, suporte ao usuário e filtros para facilitar a navegação, além de orientação clara sobre o funcionamento do sistema de lances, garantindo experiência intuitiva e eficiente.
